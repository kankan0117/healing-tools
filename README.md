## 余地

[打开「余地」](https://kankan0117.github.io/healing-tools/)

> 在情绪与行动之间，给自己留一点判断的空间。

当你因为被否定、被冷落或被不友好对待，而开始自我怀疑、反复内耗或想要冲动回应时，“余地”陪你暂时停下来，重新区分事实、想法、情绪与责任。

### 什么时候可以打开它？

- 被批评或否定之后
- 因他人的态度变化而不安时
- 人际冲突后反复内耗时
- 因失误而过度责怪自己时
- 准备冲动回应之前
